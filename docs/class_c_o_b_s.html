Detailed Description

A Consistent Overhead Byte Stuffing (COBS) Encoder.

Consistent Overhead Byte Stuffing (COBS) is an encoding that removes all 0 bytes from arbitrary binary data. The encoded data consists only of bytes with values from 0x01 to 0xFF. This is useful for preparing data for transmission over a serial link (RS-232 or RS-485 for example), as the 0 byte can be used to unambiguously indicate packet boundaries. COBS also has the advantage of adding very little overhead (at least 1 byte, plus up to an additional byte per 254 bytes of data). For messages smaller than 254 bytes, the overhead is constant.
